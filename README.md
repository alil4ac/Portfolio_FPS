# 언리얼 게임개발 포트폴리오  
## 게임 데이터 흐름 제어 및 관리  

---

### 개요  

**주요 장르**  
- FPS  
- 제한적 TPS(Back View)  
- 루트 슈터  

**모티브**  
- 데스티니 가디언즈  

**핵심 요약**  
- **캐릭터 빌드**:  
  - 데스티니 가디언즈의 핵심은 하나의 캐릭터에 여러 유형(능력, 무기, 장비 등)이 결합되어 하나의 캐릭터 빌드를 구성하는 점에 있음.  

- **시너지 시스템**:  
  - 능력, 무기, 장비 등에서 동일한 효과나 특성을 조건부나 효과로 공유함.  
  - 하나의 효과 발동 시 여러 시스템에서 동시에 효과가 발동하며, 강력한 시너지를 보여줌.  

- **무기 시스템**:  
  - 랜덤한 특성이 여러 무기에 공유되거나 고유한 무기의 전용 특성이 존재.  

- **에디터 구현 목표**:  
  - 이러한 특징을 에디터 상에서 효과적으로 확장 및 편집 가능하게 하기 위한 커스텀 에디터를 구현.

# 게임 키워드

게임의 여러 곳에서 사용되거나 공유될 수 있음. 특정 키워드는 다른 시스템(무기, 능력 등)의 조건으로 사용될 수 있음.

## A. 공용 키워드

### 1. 적 처치
- 특정 적 처치 시 발동하는 키워드.

### 2. 적 공격
- 특정 적 공격 시 발동하는 키워드.

### 3. 재장전
- 현재 무기를 재장전할 경우 발동하는 키워드.

### 4. 보호막 (플레이어블 전용)
- 임시로 캐릭터에 부여되는 추가 생명력.

### 5. 속성 보호막 (적 전용)
- 특정한 속성에 의해 파괴될 수 있는 보호막.
- 속성 보호막 파괴 이전에 대상이 받는 피해가 20% 감소.
- 보호막 속성과 일치하는 속성 피해는 해당 피해 감소를 무시.
- 보호막 속성과 일치하는 피해로 보호막 파괴 시, 일정 시간 동안 대상이 받는 해당 속성 피해 증가.
- 보호막 파괴 이후 일정 시간 뒤 빠르게 보호막 재생.

## B. 속성 구분

### 물리
- 물리 무기만 해당.
- 보호막이 파괴된 대상에게 피해량이 10%증가.

### 태양
- 태양 속성 무기 및 태양 능력이 해당.
- 피해 강화 및 회복 효과 보유.

### 전기
- 전기속성 무기 및 전기 능력이 해당.
- 대상을 경직 시키거나 광역 피해를 가함.

### 공허
- 공허속성 무기 및 공허 능력이 해당.
- 보호막 및 능력 재생속도 증가.
- 대상에 약화효과 부여 가능.

## C. 태양 속성 키워드

### 1. 치료
- 0.1초간 체력 60HP를 즉시 치유.
- 발동 후 1초의 쿨다운 존재.

### 2. 회복
- 체력을 지속적으로 치유.
- 버프 기본 지속시간은 5초, 최대 지속시간은 15초.

### 3. 광채
- 장착 중인 모든 무기의 기본 공격력이 10초간 25% 증가하고 유효 사거리가 증가.

### 4. 소각<a id="소각"></a>
- 부여된 대상은 1.5초 동안 0.5초마다 태양 속성 피해를 받음.
- 피해량은 스택에 비례해 증가.
- 소각이 50 이상 부여되면 대상에게 점화가 발생.

### 5. 점화
- 발동 시 대상을 중심으로 태양 속성 폭발 피해 발동.
- 해당 피해는 거리별 피해 감소가 적용되지 않음.
- 발동 후 대상에게 부여된 소각 스택 전부 제거.
- 점화가 발동한 대상은 2초간 점화 재발동 불가 (소각 스택은 중첩 가능).
- 소각을 부여한 능력이나 무기가 강화될 경우, 점화 또한 피해량 증가.

## 캐릭터 디테일  

### 1. 캐릭터의 능력  
- **이동**: 전진, 후진, 측면 이동, 점프
  
- **공격**:  
  - 총기 사격
  - 근접 무기 공격
    
- **스킬**:  
  - 투척물  
  - 근접 능력 
  - 특수 능력  
  - 궁극기
    
- **상세 설명**
  - 캐릭터의 스킬은 스탯의 수치에 따라 강화될 수 있습니다.
  - 각 능력은 선택한 하위 능력에 따라 효과 및 타입이 변경되거나 추가될 수 있습니다.

### 2. 캐릭터의 스탯  

캐릭터는 총 6종의 스탯을 보유합니다:  

- **힘** 
- **지능**  
- **의지**  
- **생명력**  
- **회복력**  
- **기동력**  

### 3. 캐릭터의 능력 교체  
- 각 능력마다 파생되는 여러 종류가 존재합니다.  
- 플레이어는 각 능력마다 하나를 선택하여 **캐릭터 빌딩**을 구성할 수 있습니다.  

### 4. 캐릭터의 스탯 증감 및 효과  
- 캐릭터의 스탯은 **장착 장비**에 의해서만 영향을 받습니다.  
- 특정 스탯이 일정 수치 이상일 경우:  
  - 특정 능력이 강화됩니다.  
  - 쿨다운이 감소합니다.  

### 5. 캐릭터의 장착 무기  
캐릭터는 총 3개의 무기 슬롯을 보유합니다:  

1. **1번 슬롯**: 물리 무기  
   - 특별한 속성이 없는 무기군
     
2. **2번 슬롯**: 속성 무기  
   - 3개의 속성 중 하나가 부여된 무기군
     
3. **3번 슬롯**: 파워 무기  
   - 피해량이 높고 탄약 수가 적은 무기군  

### 6. 캐릭터의 장착 장비  
- 캐릭터는 총 5개의 장비 슬롯을 보유합니다:  
  - 머리  
  - 장갑  
  - 갑옷  
  - 다리  
  - 악세서리
    
- 각 장비의 스탯 총합이 **캐릭터의 스탯**으로 반영됩니다.

### 7. 캐릭터의 패시브  
- 캐릭터의 패시브는 **상**과 **조각**으로 구성됩니다.  
  - **상**:  
    - 유니크한 추가 능력을 부여하거나, 기존 액션에 추가 능력을 더합니다.  
  - **조각**:  
    - 기존 능력을 소폭 강화하거나 효과의 범위를 확대하는 가벼운 능력을 제공합니다.  
- **장착 제한**:  
  - 상과 조각 모두 장착 가능 개수에 제한이 있습니다.
